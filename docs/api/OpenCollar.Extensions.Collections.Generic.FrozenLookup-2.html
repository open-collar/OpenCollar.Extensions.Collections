    Class FrozenLookup<TKey, TValue>

    A class that can hold a fixed set of values with an efficient key-based means of retrieving the values. This class cannot be inherited.

    Inheritance

    Syntax
    public sealed class FrozenLookup<TKey, TValue> : IFrozenLookup<TKey, TValue>, IComparable<IFrozenLookup<TKey, TValue>>, IEquatable<IFrozenLookup<TKey, TValue>>, IComparable where TKey : IComparable<TKey>, IEquatable<TKey>
    Type Parameters
    Name Description
    TKey

    The type of the key.
    TValue

    The type of the value.
    Remarks

    This class is quicker and more memory efficient than a standard dictionary, at the cost of not being able to add new items after construction.
